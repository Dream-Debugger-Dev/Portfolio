# 🚀 Anshik Kewat | Portfolio Website

Welcome to the polished personal portfolio for **Anshik Kewat** — a modern static web experience with neon sci-fi styling, smooth animations, and clean responsive design.

> **Built for:** showcasing skills, projects, contact flow, and personal branding in a sleek developer portfolio.

---

## ✨ What’s Inside

- **Single-page responsive portfolio** built with clean HTML, CSS, and JavaScript.
- **Animated hero section** with particle effects, mouse-driven parallax, and dynamic typewriter text.
- **Interactive skills and counters** with scroll-triggered animation.
- **Contact form integration** using Formspree, plus live validation and feedback.
- **Modern navigation** with sticky header, mobile menu overlay, and active scroll highlighting.
- **Optimized favicon setup** inside `assets/favicons/` for cross-browser support.
